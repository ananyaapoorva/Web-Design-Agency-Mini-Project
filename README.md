# Creative Design Agency — Landing Page

A simple, responsive landing page built with vanilla HTML and CSS to practice
layout, typography, and design fundamentals.

## Features
- Clean hero section with custom typography (Google Fonts – Poppins)
- Two-column feature cards (Beauty / Construction)
- Fully responsive layout
- No frameworks or build tools — just HTML & CSS

## Tech Stack
- HTML5
- CSS3
- Google Fonts
